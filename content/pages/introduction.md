# UI Catalog

A living catalog of daily UI concepts, built as a shadcn-compatible registry.

## What is this?

This is a component registry where each entry is a standalone UI concept — buttons, cards, layouts, animations, interactions — designed, built, and documented one day at a time.

Every component is:

- **Copy-pasteable** — install via `npx shadcn add`
- **Composable** — built on shadcn primitives
- **Documented** — each concept has its own wiki page with usage examples

## How to use

Browse the sidebar to explore components by category. Each page shows a live preview, the source code, and installation instructions.

## Installing a component

```bash
npx shadcn add https://ui-catalog.dev/r/component-name.json
```
